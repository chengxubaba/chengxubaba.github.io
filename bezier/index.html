在二阶贝塞尔曲线中，已知三点恒定（P0,P1,P2），设定在P0P1中的点为Pa，在P1P2中的点为Pb，Pt在PaPb上的点，这三点都在相同时间t内做匀速运动。

由公式(1)可知
Pa=P0+(P1−P0)t=(1−t)P0+tP1(2)P_a= P_0 + (P_1 - P_0)t = (1-t)P_0 + tP_1\quad(2)P​a​​=P​0​​+(P​1​​−P​0​​)t=(1−t)P​0​​+tP​1​​(2)
Pb=P1+(P2−P1)t=(1−t)P1+tP2(3)P_b= P_1 + (P_2 - P_1)t = (1-t)P_1 + tP_2\quad(3)P​b​​=P​1​​+(P​2​​−P​1​​)t=(1−t)P​1​​+tP​2​​(3)
Pt=Pa+(Pb−Pa)t=(1−t)Pa+tPb(4)P_t= P_a + (P_b - P_a)t = (1-t)P_a + tP_b\quad(4)P​t​​=P​a​​+(P​b​​−P​a​​)t=(1−t)P​a​​+tP​b​​(4)

将公式 (2) (3) 代入公式 (4) 中，可得：
Pt=(1−t)Pa+tPbP_t= (1-t)P_a + tP_bP​t​​=(1−t)P​a​​+tP​b​​
=(1−t)[(1−t)P0+tP1]+t[(1−t)P1+tP2]= (1-t)[(1-t)P_0+tP_1]+t[ (1-t)P_1 + tP_2]=(1−t)[(1−t)P​0​​+tP​1​​]+t[(1−t)P​1​​+tP​2​​]
=(1−t)2P0+(1−t)tP1+(1−t)tP1+t2P2= (1-t)^2P_0+(1-t)tP_1 + (1-t)tP_1 + t^2P_2=(1−t)​2​​P​0​​+(1−t)tP​1​​+(1−t)tP​1​​+t​2​​P​2​​
=(1−t)2P0+2(1−t)tP1+t2P2t∈[0,1](5)= (1-t)^2P_0+2(1-t)tP_1+ t^2P_2\quad t\in[0,1]\quad (5)=(1−t)​2​​P​0​​+2(1−t)tP​1​​+t​2​​P​2​​t∈[0,1](5)
